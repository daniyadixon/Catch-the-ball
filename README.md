# Catch the Ball Game

Catch the Ball is a simple and interactive game developed using **Python**, where the player controls a catcher to collect falling balls and score points.  
The game is designed to demonstrate basic game logic, event handling, and real-time interaction in Python.

---

## Features
- Simple and intuitive gameplay
- Real-time score tracking
- Increasing difficulty over time
- Keyboard-controlled player movement
- Beginner-friendly Python project

---

## Technologies Used
- **Python**
- `pygame` (if used)
- Basic Object-Oriented Programming concepts
